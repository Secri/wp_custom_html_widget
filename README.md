Plugin permettant d'injecter du code HTML personnalisé dans les zones réservées aux widgets.
Possibilité de créer plusieurs blocs HTML distincts.
L'affichage en front est conditionné à l'utilisation de taxonomie, donc vous pouvez ajouter vos widgets dans toutes les zones et gérer ensuite leur affichage avec une taxonomie dédiée.
Gère la correspondance stricte avec une seule taxonomie ou avec l'ensemble des enfants d'une taxonomie mère.
